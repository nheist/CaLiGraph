CaLiGraph v1.0

CaLiGraph is a large semantic knowledge graph. It has a rich ontology which is compiled from the DBpedia ontology, and Wikipedia Categories & Listpages. The ontology is enriched with fine-grained value restrictions on its classes that are learned through the Cat2Ax approach. A large amount of CaLiGraph's resources are extracted from Wikipedia Listpages by combining the ontological information with machine learning.

CaLiGraph v1.0 Release

October 1, 2019

The first version of CaLiGraph is released and the website is launched.

On October 29, 2019 the paper Uncovering the Semantics of Wikipedia Categories, which describes the foundations used for the extraction of CaLiGraph, is presented at the International Semantic Web Conference.

CaLiGraph is created and provided by the Data and Web Science Group, University of Mannheim.
